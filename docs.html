一言 API 文档

一言 API 提供了随机名言、文本或图片的获取功能，以及内容提交接口。

认证方式

除了获取之外不需要认证，其余请求需要在请求头中包含以下认证信息：

  • X-Client-ID: 客户端ID
  • X-App-ID: 应用ID
  • Authorization: Bearer token (格式: Bearer your_token)

API 端点

获取随机内容

GET /

返回一条随机的内容（文本或图片）。

请求参数

参数 类型 说明
docs boolean 设置为true可返回HTML格式的文档页面

代码示例

JavaScript (Fetch)
fetch('https://api-yiyan.dfggmc.top/')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
PHP
$url = 'https://api-yiyan.dfggmc.top/';
$response = file_get_contents($url);
$data = json_decode($response, true);
print_r($data);
Python

print(data)

获取所有内容

GET /?all=true

返回所有非隐藏内容，支持分页。

请求参数

参数 类型 说明
all boolean 必须设置为true
page integer 页码，默认为1
docs boolean 设置为true可返回HTML格式的文档页面

提交新内容

POST /

提交新的内容（文本或图片）。

请求格式

支持两种提交方式：

  1. JSON格式：用于提交文本内容
  2. 表单格式：用于提交图片内容

JSON 请求参数

参数 类型 必填 说明
content string 是 内容文本
user_name string 是 用户名
content_type string 否 内容类型，默认为"text"
quote_source string 否 来源描述

表单请求参数

参数 类型 必填 说明
content string 是 内容描述
user_name string 是 用户名
content_type string 是 必须设置为"image"
image_file file 是 图片文件
quote_source string 否 来源描述

代码示例

JavaScript (Fetch) - JSON提交
const data = {
    content: "这是一条新名言",
    user_name: "testuser",
    quote_source: "名人名言"
};

fetch('https://api-yiyan.dfggmc.top/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-Client-ID': 'your_client_id',
        'X-App-ID': 'your_app_id',
        'Authorization': 'Bearer your_token'
    },
    body: JSON.stringify(data)
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
PHP - JSON提交
$url = 'https://api-yiyan.dfggmc.top/';
$data = [
    'content' => '这是一条新名言',
    'user_name' => 'testuser',
    'quote_source' => '名人名言'
];

$options = [
    'http' => [
        'header'  => "Content-type: application/json\r\n" .
                     "X-Client-ID: your_client_id\r\n" .
                     "X-App-ID: your_app_id\r\n" .
                     "Authorization: Bearer your_token\r\n",
        'method'  => 'POST',
        'content' => json_encode($data),
    ],
];

$context  = stream_context_create($options);
$result = file_get_contents($url, false, $context);
print_r(json_decode($result, true));
Python - JSON提交
import requests
import json

url = 'https://api-yiyan.dfggmc.top/'
headers = {
    'Content-Type': 'application/json',
    'X-Client-ID': 'your_client_id',
    'X-App-ID': 'your_app_id',
    'Authorization': 'Bearer your_token'
}
data = {
    'content': '这是一条新名言',
    'user_name': 'testuser',
    'quote_source': '名人名言'
}

response = requests.post(url, headers=headers, data=json.dumps(data))
print(response.json())

PHP - 图片上传
$url = 'https://api-yiyan.dfggmc.top/';
$data = [
    'content' => '这是一张图片描述',
    'user_name' => 'testuser',
    'content_type' => 'image',
    'image_file' => new CURLFile('/path/to/image.jpg')
];

$headers = [
    'X-Client-ID: your_client_id',
    'X-App-ID: your_app_id',
    'Authorization: Bearer your_token'
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, $data);
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);

print_r(json_decode($response, true));
Python - 图片上传
import requests

url = 'https://api-yiyan.dfggmc.top/'
headers = {
    'X-Client-ID': 'your_client_id',
    'X-App-ID': 'your_app_id',
    'Authorization': 'Bearer your_token'
}
files = {
    'image_file': open('image.jpg', 'rb')
}
data = {
    'content': '这是一张图片描述',
    'user_name': 'testuser',
    'content_type': 'image'
}

response = requests.post(url, headers=headers, files=files, data=data)
print(response.json())

内容类型

类型 说明
text 文本内容
image 图片内容

图片上传限制

  • 最大文件大小：5MB
  • 支持的图片格式：JPEG, PNG, GIF, WebP
注意：通过API提交的内容默认会被标记为隐藏，需要管理员审核后才能公开显示。
